Hébergement virtuelle Login/logout
  • HostDebian Login/logout

    Ce projet a impliqué la mise en place d'une application web de connexion sur un serveur fonctionnant sous Debian, une distribution Linux réputée pour sa stabilité et sa sécurité. L'objectif était de déployer une application interactive permettant aux utilisateurs de se connecter à leur compte, le tout relié à une base de données MySQL pour la gestion des informations d'identification. La particularité de ce projet réside dans l'utilisation de l'invite de commande pour toutes les étapes, depuis la configuration du serveur jusqu'à la gestion de la base de données.

    Technologies et outils utilisés :

    - Debian : Comme système d'exploitation serveur, choisi pour sa fiabilité et sa compatibilité avec de nombreuses applications web.
    - MySQL : Système de gestion de base de données pour stocker et gérer les données des utilisateurs de manière sécurisée.
    - Apache : Serveur web utilisé pour servir les pages de l'application web aux utilisateurs.
    - PHP :Langage de script côté serveur pour développer la logique d'authentification et la connexion à la base de données.
    - Invite de commande : Utilisée pour l'installation des paquets, la configuration du serveur, la gestion de la base de données, et le déploiement de l'application.

    Déroulement du projet :

    - Configuration du Serveur : Installation et configuration d'un serveur web (Apache/Nginx) et du système de gestion de base de données MySQL sur Debian en utilisant l'invite de commande.
    - Déploiement de l'Application : Transfert des fichiers de l'application web (HTML, CSS, PHP) vers le serveur via SSH, en utilisant des commandes pour naviguer dans le système de fichiers et placer les fichiers dans le répertoire approprié du serveur web.
    - Configuration de la Base de Données : Création et configuration de la base de données MySQL pour l'application, incluant la création de tables et l'ajustement des paramètres de sécurité, le tout effectué via l'invite de commande.
    - Test et Validation : Accès à l'application via un navigateur web pour tester les fonctionnalités de connexion, s'assurer de la communication réussie entre l'application et la base de données, et vérifier la sécurité de l'ensemble du système.
