Projet fais sous Java
  • Calculatrice

    Ce projet consiste en la création d'une calculatrice qui permet d'effectuer des opérations de base telles que l'addition, la soustraction, la multiplication et la division. L'interface utilisateur, offre une expérience intuitive et facile à naviguer pour l'utilisateur.

    Technologies utilisées :

    - HTML : Pour structurer l'interface de la calculatrice, incluant l'écran d'affichage des résultats et les boutons pour les chiffres et les opérations.
    - CSS : Utilisé pour styliser l'apparence de la calculatrice, offrant un design attrayant avec un dégradé de couleurs en arrière-plan, des boutons esthétiquement plaisants, et un écran clair pour l'affichage des résultats.
    - JavaScript : Le cœur fonctionnel du projet, permettant la logique de calcul. Il gère les entrées de l'utilisateur, effectue les calculs en fonction de l'opérateur sélectionné et met à jour l'écran avec le résultat des opérations.

    Fonctionnalités principales :

    - Opérations basiques : Addition, soustraction, multiplication, et division.
    - Effacement : Un bouton 'C' permet d'effacer tous les calculs et de réinitialiser l'écran à zéro.
    - Correction : La possibilité d'effacer le dernier chiffre entré.
    - Calcul instantané : Les résultats sont mis à jour immédiatement à l'écran après chaque opération.
